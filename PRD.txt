MOCK MATE - Product Requirements Document
Product Overview
Product Name

Mock Mate

Product Description

Mock Mate is an AI-powered mock interview platform designed to help students and job seekers practice interviews in a realistic conversational environment.

The platform simulates interview sessions where users can interact with an AI interviewer, answer questions, and receive structured feedback to improve performance.

The initial version focuses on providing accessible, realistic, and role-specific interview practice without requiring expensive coaching platforms.

Problem Statement

Many students and early professionals struggle with interviews because:

they lack real interview practice
they get nervous during live interviews
they don’t know how well they are performing
existing mock interview platforms are expensive
generic interview question lists do not simulate real interaction

This creates a gap between theoretical preparation and real interview performance.

Goal

Build a web platform that allows users to:

practice mock interviews with AI
simulate realistic interview conversations
receive feedback after sessions
improve confidence and communication skills
Target Audience
Primary Users
Students
preparing for internships

Early Professionals

preparing for technical interviews

Functional Requirements
Authentication

System must:

allow secure user registration
support login/logout
persist authenticated sessions
User Profiles

System must:

store username
store profile image/avatar
Interview Engine

System must:

generate interview questions dynamically
maintain chat context
provide realistic follow-up questioning
conclude sessions cleanly
Feedback Engine

System must:

evaluate user responses
generate structured feedback
assign performance score
Data Storage

System must store:

user profile data
interview history
feedback summaries
performance scores
